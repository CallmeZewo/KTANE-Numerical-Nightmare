On the Subject of Numerical Nightmare

Why isn’t this stupid thing working!?!?

  • The symbols on the right display update on each solve of a non-ignored module. The current stage is shown on the bottom display.
  • Once all other non-ignored modules have been solved, the display will go blank and await your input.
  • Each symbol has a designated value, listed in the table below.
  • If an incorrect number is entered, the display will show the corresponding stage again.
  • A stage is invalid if one of the following conditions is met:
    • The stage display differs from the stage it should show.
    • At most one symbols is shown on the right display.
  • For invalid stages, refer to “Fixing a module”.

Calculating current valid stage’s value

If the stage number is a prime number:

Multiply the battery count with the sum of the symbol values and stage number.

If the stage number is a square number:

Add the values of both symbols and the last faulty shown stage number. If the stage has not been faulty up to this point only use the symbols sum.

Otherwise:

Take the product of both symbols.

Note down the least significant digit as your result.

Fixing a Module

If your module is faulty, open it using the small button on the bottom right of the numpad. This button also serves as the submit button for your fix and will close the module again. Locate the only correct table that corresponds to the issue with the module.

Wicked Wires: Only stage displayed is faulty
  • Use the alphanumeric position of the first letter in the serial number and add the faulty displayed stage.
  • Apply modulo 3, and add 1 to get the alphanumerical position of your Wire connection.
  • Add the 3 valid Stage numbers preceding this one and multiply thr result by the faulty displayed stage number.
  • Apply modulo 3, and add 1 to get the number position of your Wire connection.
  • Pressing a number and a letter creates a connection between them.
  • The connection will swap depending on yout input, only one connection can be made at once.
  • The fault will be resolved once the correct wire connection is made and submitted.

Diabolical Dials: At most one symbol is displayed on a correct stage
  • Multiply the values of all symbols from the last valid stage and Apply modulo 8 for the position of the first dial.
  • Add the values of all symbols from the last valid stage and apply modulo 8 for the second dials position.
  • To determine the third dial position, add the total indicator count to the last faulty shown stage number, then apply modulo 8.
  • Pressing a dial will rotate it by 45°.
  • The calculated number represents the number of presses for each dial
  • The fault will be resolved once each dial is in the right position before submitted.
Menacing Microchip: Faulty stage display with at most one symbol shown
  • Add the faulty stage number to the value of the faulty symbol. If no faulty symbol is present, use 0.
  • Apply modulo 20 to that number, add 1, and press the corresponding pin.
  • If the first pressed pin is even, multiply its position by 5, apply modulo 20, and add 1 to determine the second pin.
  • Otherwise, multiply its position by the battery count, apply modulo 20, and add 1 to determine the second pin.
  • Find the absolute difference between the first two pin positions to determine the third pin.
  • The microchip has 20 pins, starting from the top-left pin as position 1, moving clockwise.
  • The fault will be resolved once all necessary inputs are made and submitted.

Final Input

Enter each stage value in order. If a value is the same as the previous stage, press the skip button instead.
